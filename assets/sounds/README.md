# サウンド素材

このディレクトリには以下のサウンドファイルを配置します。

## 必要なファイル

1. **shoot.ogg** - 武器発射音
   - 推奨: 短い「ピュッ」という音
   - 長さ: 0.1-0.3秒

2. **hit.ogg** - 敵へのヒット音
   - 推奨: 「パシッ」という打撃音
   - 長さ: 0.1-0.2秒

3. **explosion.ogg** - 爆発音
   - 推奨: 「ドカーン」という爆発音
   - 長さ: 0.3-0.5秒

4. **levelup.ogg** - レベルアップ音
   - 推奨: 明るいファンファーレ
   - 長さ: 0.5-1.0秒

5. **pickup.ogg** - アイテム取得音
   - 推奨: 「ピロリン」という効果音
   - 長さ: 0.2-0.4秒

6. **ui_click.ogg** - UIクリック音
   - 推奨: 短い「カチッ」という音
   - 長さ: 0.05-0.1秒

## 音声ファイル形式

- **推奨フォーマット**: OGG Vorbis
- **サンプルレート**: 44.1kHz
- **ビットレート**: 128kbps以上

## サウンド素材の入手方法

### 無料素材サイト
- **freesound.org** - CC0/CC-BYライセンスのSE
- **opengameart.org** - ゲーム向けフリー素材
- **sonniss.com** - 無料サウンドパック（年次）

### 自作ツール
- **jsfxr** (https://sfxr.me/) - ブラウザで8bitサウンド生成
- **ChipTone** - レトロゲーム風SE作成
- **BFXR** - jsfxrの拡張版

### Godot内蔵ツール
- AudioStreamGenerator - プログラム生成
- AudioStreamSample - 波形編集

## 注意事項

現在、これらのファイルは存在しなくてもエラーになりません。
AudioManagerは存在しないファイルを自動的にスキップします。

サウンドファイルを追加したら、Godotエディタで自動的に認識されます。
